# Lab 6: Dungeon Editor for Balagur Fate 3

## Описание
Редактор подземелья для RPG игры, позволяющий моделировать базовые локации и расставлять на них NPC.

## Требования
- C++20
- CMake 3.16+
- Google Test (для тестов)

## Сборка
```bash
mkdir build
cd build
cmake ..
make
```

## Запуск
```bash
./lab6_app
```

## Тесты
```bash
cd build
ctest
```

## Функциональность
1. Добавление NPC (Knight, Pegasus, Dragon) с координатами и именем
2. Сохранение/загрузка NPC из файла
3. Печать списка всех NPC
4. Боевой режим с указанием дальности

## Правила боя
- Пегас никого не трогает
- Дракон ест пегаса
- Рыцарь убивает дракона

## Паттерны проектирования
- **Factory**: `NpcFactory` - создание NPC без знания конкретных классов
- **Visitor**: `BattleVisitor` - обход NPC для проведения боевых действий
- **Observer**: `FileLogger` и `ConsoleLogger` - логирование событий о убийствах
